## 我们在做什么（一句话）
残酷学分：把“残酷共学”的活动积分上链，做成有团队协作增益与退出惩罚的游戏化积分系统。

## 开始开发

### Fork
1. Fork 一份代码到你的代码库
2. 改好之后，在 `https://github.com/machiwachi/intensivesquad` 创建 Pull Request
3. base 选 main 就好
4. 群里叫 @wachi

### 启动
```
$ git clone git@github.com:machiwachi/intensivesquad.git
$ cd intensivesquad
$ npm i
$ npm run dev
```
